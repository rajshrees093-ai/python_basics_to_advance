# Python Basics to Advance 🚀

A structured collection of Python programs covering concepts from beginner to advanced levels.

## 📚 Topics Covered

### Beginner Level
- First Python Program
- Variables and Data Types
- Type Casting
- User Input Handling
- Basic Arithmetic Operations

### Current Programs
| File | Description |
|--------|-------------|
| `firstprogram.py` | Introduction to Python and basic output |
| `variable.py` | Understanding variables and data types |
| `typecasting.py` | Type conversion examples |
| `user_input_example.py` | Taking input from users |
| `area_of_rectangle.py` | Calculate area of a rectangle using user input |

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/rajshrees093-ai/python_basics_to_advance.git
```

2. Navigate to the project folder

```bash
cd python_basics_to_advance
```

3. Run any Python file

```bash
python filename.py
```

Example:

```bash
python area_of_rectangle.py
```
